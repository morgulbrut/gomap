# goMap

## Setup

### From source

Good old `go get -u github.com/morgulbrut/gomap` does the trick here.

### Binary

Just drop the binary according to your system somewhere in the path. 

### Settings

`map.toml` sets up the map. `template.html` is the template for the map.
